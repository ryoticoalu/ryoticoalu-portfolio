## Tech stack

- **Vanilla HTML / CSS / JS** — single-file site (`index.html`), no build step
- **Fonts**: Audiowide, VT323, Space Mono, Major Mono Display (Google Fonts)
- **Hosting**: Cloudflare Pages
- **Theme**: Neo-vaporwave dark, animated gradients, scanlines, noise overlay

## Local preview

Any static-file server works. Simplest options:

```powershell
# Python 3
python -m http.server 8000

# Node (if installed)
npx serve .
```

Then open `http://localhost:8000`.
